# 项目配置

## GLM-4.7 自动分析集成（强制启用）

**重要规则：每次用户提出任务时，必须先调用 GLM-4.7 进行分析，无一例外。**

### 调用方式
```bash
python "e:\Cousor_work\novel-ai-studio\scripts\glm-cli.py" -p "你的分析请求"
```

### 强制工作流程
**每次收到用户任务后，必须按以下顺序执行：**

1. **第一步（必须）**：调用 GLM 分析
   ```bash
   python "e:\Cousor_work\novel-ai-studio\scripts\glm-cli.py" -p "分析任务：[用户的任务描述]"
   ```

2. **第二步**：展示 GLM 分析结果

3. **第三步**：评估并补充 GLM 的分析

4. **第四步**：开始实施具体方案

### 排除情况（以下情况不调用 GLM）
- 用户只是在闲聊或打招呼
- 用户询问简单问题（如"这个文件在哪"）
- 用户明确说"不需要 GLM 分析"

### 示例
用户输入：`帮我做一个用户登录页面`

2. 展示 GLM 返回的分析结果
3. 评估分析并补充自己的建议
4. 开始实施
