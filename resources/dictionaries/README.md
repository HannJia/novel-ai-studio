# NovelAI Studio 敏感词库

## 说明

本目录存放敏感词库文件，用于导出发布前的敏感词检测功能。

## 目录结构

```
dictionaries/
├── index.json          # 词库索引
├── qidian-words.txt    # 起点中文网禁用词（预留）
├── zongheng-words.txt  # 纵横中文网禁用词（预留）
└── README.md           # 本文件
```

## 词库格式

### 文本格式（.txt）

每行一个敏感词：

```
敏感词1
敏感词2
敏感词3
```

### JSON 格式

```json
{
  "words": [
    {
      "word": "敏感词",
      "replacement": "替换词（可选）",
      "level": "forbidden|warning"
    }
  ]
}
```

## 使用方式

1. 系统自动加载词库
2. 导出文稿时自动检测
3. 高亮显示敏感词位置
4. 提供替换建议

## 注意事项

- 敏感词库需要定期更新
- 不同平台的敏感词要求不同
- 用户可以自定义添加词汇
- 词库仅供参考，最终以平台审核为准
